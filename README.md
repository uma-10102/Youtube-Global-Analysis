# Youtube-Global-Analysis
An end-to-end exploratory data analysis project on YouTube video data, focused on uncovering engagement patterns and performance drivers using Python, Pandas, and data visualization techniques.

# Project Overview
This project presents an exploratory data analysis (EDA) of YouTube video data to understand content performance and audience engagement. By examining metrics such as views, likes, comments, and engagement ratios, the analysis uncovers patterns that influence video success.

# Objectives

* Explore and clean YouTube video data
* Analyze key performance and engagement metrics
* Identify trends and relationships between views, likes, and comments
* Visualize insights for better interpretability

# Tech Stack

 **Pandas** – data manipulation
 **NumPy** – numerical analysis
 **Matplotlib / Seaborn** – data visualization
 **Jupyter Notebook** – interactive analysis

# Analysis Highlights

* Data preprocessing and validation
* Descriptive statistical analysis
* Engagement-based comparisons across videos
* Visual exploration of performance trends

# Key Takeaways

* Improved understanding of social media engagement metrics
* Practical exposure to real-world exploratory data analysis
* Strengthened data visualization and storytelling skills
